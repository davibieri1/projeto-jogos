# projeto-jogo
alunos: Davi Lasmar Bieri e Arthur aleixo
jogo em C usando estruturas de seleção e repetição
3 mini-jogos:

-perguntas e respostas: 
é um jogo de conhecimentos gerais onde os jogadores enfrentam uma série de desafios intelectuais, precisando escolher a alternativa correta ou formular respostas precisas sobre temas variados para acumular pontos e vencer a partida.

-cobra na caixa! : 
é um jogo de suspense e sorte no qual os participantes devem abrir recipientes ou acionar mecanismos sem saber em qual deles uma cobra de brinquedo está escondida, sendo eliminados ou perdendo a rodada aqueles que acabarem "picados" pelo bote do brinquedo.

-gousmas war : 
é um jogo de estratégia e combate em que os jogadores controlam exércitos de criaturas gelatinosas com habilidades únicas, utilizando táticas de posicionamento e ataques especiais para dominar o território adversário e eliminar as gosmas inimigas do campo de batalha.
